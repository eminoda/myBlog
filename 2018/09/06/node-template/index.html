为何选择 nunjuks？

  • mozilla 项目
  • 有完善的文档，简约设计，详细 API，还有 中文版本
  • 模板 html 风格。不像 jade 那么重视严谨的缩进
  • 包含主流模板功能。继承、过滤器、逻辑判断、循环…
  • 可支持到 IE8
