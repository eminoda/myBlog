## SEO

SEO（Search Engine Optimization），搜索引擎优化。

### 目的

利用搜索引擎的规则，提升网站 **自然排名** ，让自己的网站能更好的被用户检索到。这样有利于通过技术手段营销网站，提升品牌价值。

### 技术手段

主要的技术实现方式：**白帽** 和 **黑帽**

白帽：遵守，并利用搜索引擎的规则。让搜索引擎更好的抓取网站内容，为用户提供更优质的内容。

黑帽：与白帽相反，想法设法钻搜索引擎的漏洞，利用各种手段让质量差的网站排名优于其他站点。
